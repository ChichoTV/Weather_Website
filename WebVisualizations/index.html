Summary of Project

The purpose of this project was to analyze correlations between weather patterns and distance from the equator. OpenWeatherMap API was utilized to pull the data, python was used to clean and analyze the data, and Matplotlib was used for visualizations. Some of the factors exmained include cloudiness, wind speed, humidity, and temperature. Through the various links on this site you can discover the analysis and insights from the project.
