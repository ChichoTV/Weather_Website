Clouds Vs. Latitude

We can see that there is practically no correlation between cloud count and latitude. One noteworthy observation to be made is that the data tends to cluster around discrete values which could be a result of how cloudiness is measured (though it's possible the API we used may have used some other means of calculating cloudiness).
